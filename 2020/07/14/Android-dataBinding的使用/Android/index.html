    布局和绑定

    借助表达式语言，您可以编写表达式来处理视图分派的事件。数据绑定库会自动生成将布局中的视图与您的数据对象绑定所需的类。

    数据绑定布局文件略有不同，以根标记 layout 开头，后跟 data 元素和 view 根元素。此视图元素是非绑定布局文件中的根。例：

    在 data 里面添加变量 user 然后使用 @{} 语法写入特性属性中
